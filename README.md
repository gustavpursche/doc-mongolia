# Scroll-Reportage about the Mongolei #

## Installation ##
```bash
npm install && bower install
```
## Run ##
```bash
node server.js
```

Visit http://localhost:3000/

## Watch for Changes ##
```bash
grunt watch
```

## Build Dev Enviroment ##
```bash
grunt replace:dev
```

## Build Prodution Enviroment ##
```bash
grunt build
```
